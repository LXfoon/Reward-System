# Productivity System Web

A personal point-based productivity tracker designed to gamify daily habits and tasks.

## Features

- **Current Points Dashboard** – View your real-time point balance at a glance
- **Quick Point Addition** – One-click buttons for common tasks to log progress instantly
- **Custom Point Adjustment** – Manually add or subtract points for flexible tracking
- **Point Distribution List** – Predefined actions with point values (e.g., +20 for completing work, -10 for buying a sweet treat)
- **Point History Log** – Complete transaction history showing what was earned or spent and when

## Technologies Used

- PHP
- CSS
- MySQL
